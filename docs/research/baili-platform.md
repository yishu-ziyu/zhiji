## 三件关键事实(每件 1-3 句 + URL)

### 1. "国内运营"对应的部门
- 集团电商主要由**百丽时尚集团电商中心**运营
- 主体公司:**百丽电子商务(上海)有限公司**
- 覆盖渠道:天猫 / 京东 / 抖音 / 小红书公域 + 自播 + 私域
- 一手 URL: https://weibo.com/belleds (百丽电商官方微博主体认证)

**与"百丽国际"母公司的区别**:
- 百丽国际 = 母公司(现主做出海与运动滔搏)
- 百丽时尚集团 = 鞋服业务主体
- 招聘时两套名称并列出现,如果聊到具体岗位要先确认是哪个业务线

### 2. 数字化转型进展(腾讯/阿里合作)
- 2017 高瓴私有化后启动全价值链数字化
- 2018 与腾讯云落地"**优 MALL**"门店数字化项目
- 后续连续三年获腾讯智慧零售"全域数字化经营榜"
- 2025-04 入选虎嗅"大鲸 GenAI 落地案例 TOP10"
- 2025-08 登上央视财经《正点财经》
- 一手 URL: https://f.cfw.cn/view/c246569.html (优 MALL 首店报道)
- 未核实到阿里侧 2024-2026 新合作(信息缺口)

### 3. 近 1 年(2025-2026)关键动作
- **直播电商**为主引擎:
  - 2024 抖音超品日
  - 4700㎡ 直播基地
  - 275 人自播团队
  - 618 天猫旗舰店同比 +79%
- **跨境电商**:
  - 2026-06 启动 9610 跨境直邮(天津武清基地,预计 H2 营收 5 亿)
- **品牌汰换**:
  - 退出暇步士
  - 新增"美丽宝"
  - 接手 Allbirds 中国 10 年经营权
  - 与 Champion 母公司成立合资
- **资本动作**: 2024-03 重新递交港交所招股书
- 一手 URL: https://post.m.smzdm.com/p/a4qvvxw7/ (2026-06 武清跨境项目,二手)

## 体量与对比

| 指标 | 2017 退市时 | 当前 |
|------|----------|------|
| 私有化金额 | 531 亿港元 | - |
| 门店数 | 8000+ 直营 | 仍称 8000+ |
| 电商占线上营收 | < 7% | ~27% |
| 电商占集团总营收 | < 7% | ~30% |

**核心趋势**:电商占比从 <7% 提升到 ~30%,这是百丽过去 8 年最大的结构性变化。
