## 2026-07-17 · 上线水准与科学训练（调研结论）

### 产品 Agent 类型定义

**领域内、多步工具 + 强制 grounding + HITL 的项目情报 Agent**  
不是通用助手、不是编程 Agent、不是开放 Web Agent。

### 上线门槛（对话结论摘要）

**L0 工程红线（任一 FAIL 不可上线）**

1. 没读真实文件就下结论  
2. 有结论但点不开出处  
3. 界面无真实工具过程  
4. 模型挂了仍显示读懂  
5. 确认后重进丢失  

**L1 演示/内测**（示意阈值）：工具闭环 ≥95%、出处可点开 ≥90%、拒答诚实 ≥90%、结构完整 ≥85%、噪声不升格 ≥90% 等  

**L2 付费可用**：Faithfulness ≥0.85、Citation precision ≥0.90、危险动作 0、接受后恢复 100% 等  

**明确不采用：** 用 SWE-bench / 通用 GAIA 总分作为知几上线门槛。

### 科学训练路径（对话结论）

```text
系统硬约束 + 自建 Project Intelligence Bench
  → SFT（轨迹：工具序 + UnderstandingBody + 结构对话）
  → DPO/偏好（有证据 > 流畅胡说）
  → Process supervision / Verifier-RL（步级工具与引用可验证）
  → Owner 终裁日志回流
```

最高 ROI 三件事：

1. 锁 50–100 题自建 Bench + CI  
2. Citation/faithfulness 自动 verifier  
3. 500–2000 条改写好轨迹 SFT + 一轮诚实 DPO  

### 当前定位（对话判断）

- **P0 骨架扎实**（工具环、证据门、HITL、fail-closed、结构对话）  
- 距「陌生人可依赖日常产品」差在：**固定评测规模、跨项目统计、训练数据闭环**
