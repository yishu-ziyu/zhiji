# 主人试用手册 · 真实接入（015）

**打开：** http://localhost:3000/track/knowledge  

**你怎么用：** 按场景做左边行为，看右边是否出现。对勾自己打。

**说明：**  
- 这是 **BDD（行为）**，不是代码说明。  
- 本版 **不做** 整夹拖入自动拆成多个项目。  
- 若你一打开就已经有项目，先看场景 0B；要验「甲」必须处在 **还没有项目**（场景 0A）。

---

## 场景 0A · 第一次 / 还没有项目

| 步骤 | 当你… | 则你应看到… | ✓ |
|------|--------|-------------|---|
| 0A-1 | 打开上面网址 | 标题类似「还没有项目」；说明环境是空的、无预置项目；有 **新建项目**、**上传文件** | |
| 0A-2 | 看整页 | **没有** 一堆「像真的」的示例任务/示例项目当工作局面（除非你本机以前自己建过） | |

**过：** 诚实空/引导。  
**不过：** 一进来就像已经在跟一个编好的项目。

## 场景 0B · 打开时已经有项目

| 步骤 | 当你… | 则你应看到… | ✓ |
|------|--------|-------------|---|
| 0B-1 | 打开网址 | 进入某个项目工作台（搜索、新建菜单等可用） | |
| 0B-2 | 想验「甲」 | 需要先清到无项目，或换干净环境；否则直接做场景 3（已有项目时拖/传） | |

## 场景 1 · A3 甲：无项目时拖文件

| 步骤 | 当你… | 则你应看到… | ✓ |
|------|--------|-------------|---|
| 1-1 | 在 **还没有项目** 时，把 **单个本地文件** 拖进页面工作区 | 遮罩提示类似：**「松开后先新建并命名项目，再收下文件」** | |
| 1-2 | 松开鼠标 | **不会** 直接进项目；弹出 **「先命名项目再收文件」**；列出将收下的文件名 | |
| 1-3 | 名称留空就点创建 | 提示名称不能为空 / 创建不了 | |
| 1-4 | 填名称（可写摘要），点 **「创建并收下文件」** | 提示类似 **「项目已创建，并收下「某文件」」**；进入该项目；材料在项目里能看见 | |
| 1-5 | 看项目/材料 | 能感到材料属于 **刚建的这个项目**（不是无主漂浮） | |

**过：** 先命名确认，再收文件。  
**不过：** 无项目直接塞进未知地方；或假项目自动出现。

## 场景 2 · A3 甲：无项目时点上传

| 步骤 | 当你… | 则你应看到… | ✓ |
|------|--------|-------------|---|
| 2-1 | 无项目时点 **上传**（工具栏上传，或空状态「上传文件」） | 系统文件选择框 | |
| 2-2 | 选一个本地文件 | 与场景 1 相同：先命名弹窗，确认后再收进新项目 | |

**过：** 与拖拽同结果。  
**不过：** 上传与拖拽两套逻辑、一个跳过命名。

## 场景 3 · A1 / A2：已有项目时再加文件

| 步骤 | 当你… | 则你应看到… | ✓ |

| 3-2 | 已在某项目中，点上传选文件 | 同上，进 **当前** 项目 | |
| 3-3 | 在项目内找该材料 | 能看见；点开/列表里能对上文件名 | |

## 场景 4 · A4：接入算数（重开还在）

| 步骤 | 当你… | 则你应看到… | ✓ |
|------|--------|-------------|---|
| 4-1 | 场景 1 或 2 成功后，**刷新页面**（或关标签再开同一网址） | **仍有** 你建的项目；**仍有** 你加的文件 | |

**不过：** 刷新项目没了或文件丢了。

## 场景 7 · A5 整夹 → 项目（已纳入）

**准备：** 本地建两个文件夹，例如 `项目甲/`、`项目乙/`，各放 1～2 个文件。

| 步骤 | 当你… | 则你应看到… | ✓ |
|------|--------|-------------|---|
| 7-1 | 一次拖入**两个**顶层文件夹进工作区 | 出现对应**两个项目**（名≈夹名） | |
| 7-2 | 打开「项目甲」 | 能看见甲夹里的文件材料 | |
| 7-3 | 打开「项目乙」 | 能看见乙夹里的文件材料 | |
| 7-4 | 刷新重开 | 两项目与材料仍在 | |
| 7-5 | （可选）只拖一个夹，内仅文件无子夹 | 建**一个**项目并收齐文件 | |

**过：** 多夹 → 多项目，材料对位。  
**不过：** 只当普通多文件塞进单一未知项目；或夹名丢了。
