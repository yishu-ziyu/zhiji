## 我们的客户是谁

**核心用户**：年收入 20-100 万的"一人公司"——淘宝/抖店小商家、自媒体创业者、独立开发者、外包工作室老板。

**规模**：中国大约有 **1600 万** 一人公司/个体工商户。每人每年在运营上愿意花 **3000-8000 元** 工具费。

## 第一年财务模型（保守）

```
MAU 目标: 5,000 个
付费转化率: 10% → 500 个付费
ARPU: 150 元/月（混合）

年收入 = 500 × 150 × 12 = 90 万元
毛利率: 70%（云函数 + LLM 边际成本 < 30%）
毛利: 63 万元
```

---

## 为什么我们能赢

1. **场景深**：电商 + 团队效率两个高 ARPU 场景，明确不是聊天机器人
2. **数据飞轮**：用户的输入（商品名、会议记录）就是我们的训练数据
3. **平台成本低**：基于阿里云函数计算 FC，**0 运维**，扩容按毫秒计费
4. **生态优势**：阿里云 OPC 社区 + Next 实验室给了我们获客渠道

- **被并购**：阿里钉钉 / 字节飞书 / 京东商家版（最可能）
