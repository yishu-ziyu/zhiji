# 知识工作助手 · 产品说明

**版本：** 0.3
**日期：** 2026-07-14
**完整版：** `.ship/tasks/knowledge-mainline-20260714-174246/product/08-prd.md`

## 1. 做什么

给**知识工作者**用：

**资料检索（带来源）→ 知识卡片（能再用）→ 待办（能勾状态）**

不是记笔记软件，不是客户改约定工具。

## 2. 给谁用

- 先：你自己
- 也可讲：效率赛道上的一人公司 / 小团队知识岗

## 3. 痛在哪

1. 材料散在聊天、纪要、文档里，要时找不回
2. 找到了也**没来源**，不敢当事实
3. 有结论落不成**能做完、能验收**的下一步

## 4. 能力

| 能力 | 说明 |
|------|------|
| 检索 | 关键词 + 来源筛选；结果带来源 |
| 写入 | 手记 / 粘贴会议 / 拆目标 |
| 待办 | 待开始 → 进行中 → 已确认 → 完成 |
| 工具接口 | `/api/knowledge/mcp` |
| 存盘 | `data/knowledge/*.json` |

## 5. 怎样算做完（验收）

- 能搜到带来源的卡
- 新卡重启后还在
- 待办状态能推进并保持

## 6. 不做

- 客户改约定当产品
- 个人微信读私聊
- 把「向量库 / MCP 协议」当卖点吹满

## 7. 演示时怎么走

打开 `/track/knowledge` → 搜「检索 来源」→ 推进一条待办 → （可选）写手记或贴会议 → 说：搜得到、卡片有来源、下一步能勾掉。
