## 词怎么用

| 说法 | 意思 | 别说成 |
|------|------|--------|
| 知识工作者 | 靠检索、笔记、会议材料干活的人；先服务你自己 | 空泛「所有一人公司」 |
| 知识卡片 | 一条能以后再用、能说来源的记录 | 随便一篇长文 |
| 来源 | 会议 / 邮件 / 聊天 / 文档 / 手记 | 没有出处的「AI 总结」当事实 |
| 知识足迹 | 库图上：亮=被检索/使用，暗=未用到 | 把亮当成「更正确」；无集合校验的装饰动画 |
| 关系 / 边 | 两张依据卡之间可解释的连接；必带来源句 | 只有 id 互链、无「凭哪句」；把足迹亮暗当成关系 |
| 来源句 | 支持这条边的一句可指回文字 | 空话「相关」、不可点回的摘要 |
| 工作项 | 一件事：状态 + 负责人 + 下一步 + 时间线 | 空待办标题、没人负责 |
| 事件 / 时间线 | 评论、决定、状态变化、阻塞、结果写回 | 只在聊天里说过 |
| 状态 | 待开始 → 进行中 → 待确认 → 完成；可阻塞 | 一堆自定义状态 |
| 演示步骤 | 固定几步，方便重复演示和验收 | 金路径、黄金路径、金脚本 |
| 失败 | 结果没来源、重启丢数据、状态点不动、和记笔记软件说不清差别 | 只报个错误码 |
